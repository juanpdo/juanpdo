<h1 align="center">Hola, soy Juan Peña Dorado</h1>
<p align="center">Soy Juan Peña Dorado, Ingeniero de Edificación reconvertido al desarrollo de software y apasionado por la tecnología. Para mí, el desarrollo web no consiste en apilar código de forma automática, sino en proyectar arquitecturas limpias, escalables y eficientes, aplicando el mismo rigor de cálculo y optimización que exige la ingeniería estructural. Mi transición al desarrollo de software nace de una profunda vocación tecnológica y de la necesidad de aplicar esa mentalidad de diseño y resolución de problemas complejos a entornos digitales.

Me especializo en el ecosistema Frontend, donde combino una base sólida en lógica de programación y bases de datos con el dominio de las tecnologías web estándar. Mi enfoque se centra en el rendimiento y la optimización de las interfaces, entendiendo el funcionamiento del lenguaje nativo para exprimir al máximo las posibilidades de las bibliotecas y herramientas modernas. Mi objetivo es aportar soluciones interactivas eficaces y robustas, fusionando la disciplina de la ingeniería con la versatilidad del desarrollo de software actual.</p>
